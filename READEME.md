** Pojeto Banco De dados de um Sistema de Vendas **

Este projeto foi desenvolvido para a criação de um banco de dados relacional
aplicada a um Sistema de Vendas.

A primeira parte do projeto contem a representação do modelo do DB através de
um diagrama de entidade-relacionamento

Criação das tabelas:
  - Clientes;
  - Produtos;
  - Pedidos;
  - Fornecedores;
  - Compras;

O projeto possui um código fonte revisado e documentado.
